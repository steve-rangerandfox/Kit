# Kit — Claude Code Guide

Kit is Ranger & Fox's AI production-management system. It spans a persistent Slack Bolt service, a Next.js/Inngest service, Supabase, and external integrations including Dropbox, Frame.io, Harvest, Google Drive, and Google Calendar.

1. Do not explore the repository broadly. Read only the files required for the task.
